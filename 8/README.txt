Отчет по проекту FullTaskManager

Функциональность:
- Добавление задач с параметрами: название, описание, приоритет.
- Удаление задач по названию.
- Просмотр всех задач.

Реализовано:
1. Базовое логирование (1.1–1.3)
2. Продвинутое логирование с Serilog (2.1–2.3)
3. Трассировка операций с TraceSource и Stopwatch (3.1–3.2)
4. Обработка и логирование исключений (3.3)

Логи:
- Консоль.
- Файл logs/log.txt с ротацией.

Пример структурированного лога:
{ "Timestamp": "...", "MessageTemplate": "Добавлена задача {@Task}", ... }

Проект можно запускать как .NET Console Application.
